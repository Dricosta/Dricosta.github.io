Tecnologias que venho estudando no momento

Utilizo muitas dessas tecnologias abaixo em projetos da faculdade, avaliando sempre o que pode ter mais relevância para ser utilizado no mercado

  • Github

    Utilizo tanto para hospedagem de projetos quanto para estudo de code, o github permite que você salve seus projetos sem perder o conteúdo de atualizações anteriores, além de facilitar no trabalho em equipe

  • Scrum

    A metodologia para organização de projetos, utilizamos Scrum para controlar prazos e gerenciar o desenvolvimento do projeto, o time tem um objetivo e comprimos metas para que esse objetivo seja alcançado.

  • JavaScript

    Utilizo o JS na parte de front dos projetos para que o usuário possa ter mais interação com o site, mesmo que eu ainda esteja estudando os eventos do javascript, dou o melhor pra que possa sair algo bacana

  • PHP

    Essa é a linguaguem de servidor que utilizo, desenvolver formulario de login, utilizar parametros bem conhecidos como GET e POST isso que venho estudando no momento

  • Trello

    Utilizo esse site como gerenciador de projetos, facilita muito na orgazição do desenvolvimento, aonde cada um tem sua tarefa para cumprir, criamos pequenos objetivos e só avançamos no desenvolvimento quando todos atingirem esse objetivo.

  • PhotoShop

    Utilizo este software quando é necessario fazer alguma alteração nas imagens do projeto, as vezes colocar algum efeito, fazer um logotipo, layout ou wifreme do site.
